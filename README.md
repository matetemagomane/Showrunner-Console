# Showrunner 🎬 | TV Content Library Management Console

Showrunner is an optimized, administrative streaming platform platform built to manage media assets, catalog entries, and live broadcasting metric variables. It translates strict relational item properties into a fluid, highly accessible, custom-designed dark workspace environment.

---

## ✨ System Architecture Features

* **Secure Entry Portal:** A centered minimal access gate simulating standard workflow identity validation checkouts before granting dashboard workspace clearance.

* **Complex Data State Interaction:** Complete CRUD operations mapped dynamically via vanilla client-side logic arrays. Elements support dynamic entry initialization, deep row deletion tracking with strict verification, and a fully reactive edit subsystem.

* **Blurring Overlay Modals:** Built structural edit components using an interactive modal viewport configuration backed by soft `backdrop-filter: blur(5px)` layouts for maximum UI focus segregation.

* **Instant Filter & Search Matrix:** Event listeners bind the search field lookup mechanics with explicit action tab buttons to seamlessly parse IDs, titles, or genres concurrently while keeping visual layout structures lightweight.

---

## 🎨 Creative Theme Framework

The application implements a unique styling language avoiding standard corporate color ranges:
* **Canvas Underlay:** `#202226` (Warm Dark Slate)
* **Interactive Accents:** `#00B9C6` (Premium Turquoise Core Elements)
* **Aggregates Indicator:** Direct 4px contextual side-border accent callouts ensuring card components communicate key numbers clearly at a glance.
